# rosbag_prosser

* create workspace
#### `cd rosbag_processer/src`
#### `catkin_init_workspace`

* make file executable
#### `chmod +x rosbag_processer/src/read_bag/read_bag.py`

* bulid and run
#### `cd rosbag_processer`
#### `catkin_make`
#### `source devel/setup.bash`
#### `rosrun read_bag read_bag.py [bag] [topic]`
